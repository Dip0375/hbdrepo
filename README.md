## üìÅ Setup Instructions

### 1. Add Your Photos

Create a `photos` folder in the same directory as the HTML file and add:

**For OLED TV Slideshow:**
- `photo1.jpg`
- `photo2.jpg`
- `photo3.jpg`
- `photo4.jpg`
- `photo5.jpg`
- `photo6.jpg`

**For Left Side Frames:**
- `left1.jpg`
- `left2.jpg`
- `left3.jpg`

**For Right Side Frames:**
- `right1.jpg`
- `right2.jpg`
- `right3.jpg`

### 2. Add Background Music (Optional)

Add a file named `birthday-music.mp3` in the same directory as the HTML file.

**Where to get birthday music:**
- Download royalty-free birthday music from sites like:
  - [YouTube Audio Library](https://www.youtube.com/audiolibrary)
  - [Free Music Archive](https://freemusicarchive.org/)
  - [Incompetech](https://incompetech.com/)
- Or use any MP3 file you have

**Note:** If the music file is not found, the page will still work perfectly without music!

### 3. Open the Page

Simply open `index.html` in your web browser and click "Click to Celebrate" to start!
